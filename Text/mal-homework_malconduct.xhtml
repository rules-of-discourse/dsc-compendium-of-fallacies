Homework Malconduct

Malconducts

Basic Description

The Homework Malconduct is committed by a participant in the discussion (Person A) when zie attempts to assign another participant (Person B) the task of researching evidence and arguments that it is in fact Person A's job to present.

Razors

If Person A cites external material as the source of an argument provided, that does not qualify as an instance of the Homework Malconduct - provided that the following criteria are met:

Zie is exact as possible where in the external material Person B needs to look to find the source of the argument.
Zie references the exteranl material in addition to providing the argument in the discourse itself - not instead of it.

Furthermore, an action must not be regarded as an instance of the Homework Malconduct when it is properly an execution of the Homework Against Repetition Bonconduct, which is often the proper response to the Exhausting Repetition Malconduct.
