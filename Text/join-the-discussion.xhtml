Join the Discussion

This project can not move together at even a remotely reasonable pase as long as only one person is working on it. Very often it is difficult, if at all possible, to assure objectivity of new entries if the authorship does not have anyone to discuss the ideas with prior to entering them into this compendium.

For this reason, a discussion forum has been set up on Facebook. Yes - Facebook is far from the ideal place to host this discussion, but until a better alternative is available, it is better than nothing. Please join the discussion group on Facebook (https://www.facebook.com/groups/rules.of.discourse/) where, by participating in the discussions, you will be providing this project with much-needed help.

Of course, the location of where this discussion is hosted may change. For that reason, before you join the discussion forum, please check the latest version of this page to make sure that your information on where the discussion is going on is current.
